## 📂 Proyectos Destacados

### 🏠 Servidor casero de domótica (Con ubuntu).

*Proyecto que une infraestructura de sistemas y automatización.*

- **❓ ¿Qué hace?:** Reutilización de un portátil Sony Vaio antiguo reconvertido en un servidor Linux local de bajo coste. El servidor actúa como el DNS principal de toda mi red local, filtra el tráfico a través de **Pi-hole** para bloquear anuncios y telemetría a nivel de red antes de que las peticiones salgan al router. Todo el ecosistema está gestionado de forma visual mediante **CasaOS** o mediante SSH desde mi propio equipo.

- **🛠️ Tecnologías utilizadas:** Linux (ubuntu), Pi-Hole, CasaOS, Redes Locales (Configuración de DNS y DHCP).

- **📈 Escalabilidad:** Proximamente actualizaré este servidor a una raspberry pi para una mayor optimización y menor consumo.
